# Easy-Wi NextGen

Easy-Wi NextGen ist ein modernes Webinterface (Panel) zur Verwaltung von Services auf deinen Servern.
Das Panel arbeitet mit Agent/Runner-Komponenten zusammen, um Aufgaben zu automatisieren und
Infrastruktur zentral zu steuern.

## Was das Interface kann
- **Zentrale Verwaltung** von Hosts, Rollen und Services über ein Web-Panel.
- **Automatisierung** von Aufgaben durch den Easy-Wi Agent.
- **Game-/Service-Workflows** über den Runner für serverseitige Prozesse.
- **API-gestützte Steuerung** für Integrationen und externe Tools.

## Komponenten im Überblick
- **/core** – Symfony Webinterface und API
- **/agent** – Go-basierter Agent für verwaltete Hosts
- **/installer** – Installer für Linux/Windows
